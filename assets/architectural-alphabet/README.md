# PANACEA — Architectural Alphabet

《PANACEA — Architectural Alphabet》是一套将 Panacea Architectural Regular 字形与建筑原型逐字对应的视觉研究。

This series maps every glyph in Panacea Architectural Regular to an architectural archetype.

## Series constants / 系列统一规则

- Winter, 10:00 AM / 冬季上午 10 点
- Dense fog, no people / 大雾、无人
- Concrete industrial architecture / 混凝土工业建筑
- Wet ground / 湿地面
- Telephoto compression / 长焦压缩
- Cool neutral gray palette / 冷灰色调
- Systematic frontal architectural photography / 客观、正立面的建筑类型学摄影
- Minimal composition, large overcast sky / 极简构图、大面积阴天天空
- Architecture occupies about 80% of the visual subject / 建筑主体约占画面 80%
- No logos, vehicles, signage, text, or plant subject / 无 logo、车辆、标识、文字或植物主体
