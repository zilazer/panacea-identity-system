## Handling Rules / 使用规则

- Do not treat legacy files as current master marks. / 不要把历史文件当作当前主标识使用。
- Do not redraw the `a` icon casually; it is a core brand asset. / 不要随意重绘 `a` 图标；它是核心品牌资产。
- Any new asset should declare whether it belongs to Master Brand, Concept Layer, Archive Layer, or Icon Layer. / 任何新资产都应声明其属于主品牌、理念层、档案层或图标层。
- Vehicle sticker outputs should use matte, die-cut, no-background production logic unless a later spec explicitly says otherwise. / 除非后续规格明确说明，否则车辆贴纸输出应遵循哑光、模切、无底色的制作逻辑。
