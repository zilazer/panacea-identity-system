Panacea Architectural Type System

源于 panacea 的建筑构件式字形系统。

Not a Typeface. A System of Architectural Components. 不是一套字体，而是一套建筑构件系统。

01 / Source / 源字

The wordmark is the source.

字标不是字体的应用结果，而是整套字形语法的起点。

The lowercase system begins with five parent glyphs found inside panacea.

所有字母均从 panacea 中已有的五个母字形推演，而不是被分别设计。

02 / Components / 构件

Six components. One language.

六种有限构件，组成二十六个小写字母。

The alphabet is assembled like a structure: columns carry, beams connect, roofs set direction, and openings preserve space.

构件 / Component结构 / Form作用 / Function

改变受力方向 / Changes direction

03 / Grammar / 规则

A finite grammar.

字形的差异来自组合，不来自装饰。

Every glyph accepts the same structural agreement. The system stays recognizable even when the letter changes.

018

Base unit / 基础单位

Every coordinate resolves to the 8-unit module.

026×16

Module grid / 模块网格

Standard glyphs occupy a 48 × 128 field.

038

Stroke / 统一线宽

One weight across every structural component.

0415.9°

Measured from the original panacea wordmark.

Width / 字宽

Narrow, standard, and wide structural bays.

060

Decorative curves / 装饰曲线

Open counters and straight construction only.

Letters are assemblies.

字母是构件的不同装配结果。

The parent glyphs expose the grammar. Their parts repeat, rotate, extend, or remain open.

Twenty-six letters. Three marks.

同一网格上的二十六个字母与三个标点。

The active font renders one continuous field: a-z, comma, period, and question mark.
