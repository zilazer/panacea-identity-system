# Vehicle Sticker Specification / 车辆贴纸规格

## Objective / 目标

Create architectural micro-branding for vehicles through carefully positioned die-cut decals.

通过精确定位的模切贴纸，为车辆建立建筑化的微型品牌识别。

The goal is not car modification. The goal is subtle identity placement that feels like product detailing.

目标不是改装车，而是让品牌露出像产品细节一样安静、精确、克制。

## Production Standard / 制作标准

Recommended production:

推荐制作方式：

- die-cut vinyl / 模切车贴膜
- no background / 无底色
- matte finish / 哑光表面
- automotive-grade adhesive / 车规级背胶
- removable where possible / 尽可能可移除
- outdoor UV resistance / 户外抗 UV
- no chrome / 不使用镀铬
- no reflective film / 不使用反光膜
- no large continuous graphics / 不做大面积连续图形

## Color System / 色彩系统

Primary sticker colors:

主贴纸颜色：

| Color / 颜色 | Use / 使用场景 |
| --- | --- |
| Matte white / 哑光白 | Dark glass, black trim, dark paint / 深色玻璃、黑色饰条、深色车漆 |
| Matte silver / 哑光银 | Light body panels, metal-adjacent detail / 浅色车身、接近金属部件的细节 |
| Gunmetal gray / 枪灰 | White or light paint, quiet low-contrast placement / 白色或浅色车漆、低对比克制位置 |

Avoid red except in documented historical or special-edition contexts. The vehicle program should not depend on red as a default accent.

除非用于明确记录的历史语境或特别版本，否则避免红色。车辆系统不应默认依赖红色作为点缀。

For the user's deep metallic blue Tesla Model 3, prioritize matte silver for blue body panels, matte white for glass and black trim, and gunmetal gray only for very subtle owner-only or close-inspection details.

针对用户的深蓝金属色 Tesla Model 3，蓝色车身面优先使用哑光银，玻璃与黑色饰条优先使用哑光白，枪灰仅用于非常克制的车主可见或近距离细节。

## Placement Zones / 贴附区域

Recommended visible zones:

推荐可见区域：

- front bumper / 前保险杠
- headlight corner / 头灯角落
- fender / 翼子板
- camera housing / 摄像头外壳
- door handle / 门把手
- rear quarter window / 后三角窗
- charging port / 充电口
- rear bumper / 后保险杠
- wheel center cap / 轮毂中心盖
- wheel spoke / 轮辐
- mirror bottom / 后视镜下方
- trunk corner / 后备箱角落

Hidden zones:

隐藏区域：

- frunk / 前备箱
- door jamb / 门框内侧
- charging port interior / 充电口内部
- rear trunk interior / 后备箱内部
- center console / 中控区域
- seat rail / 座椅滑轨

Hidden placement principle: only the owner knows.

隐藏位置原则：只有车主知道。

## Placement Rules / 贴附规则

- Use point graphics only. / 只使用点状图形。
- Prefer alignment with existing panel gaps, trim lines, camera modules, or mechanical details. / 优先对齐现有钣金缝、饰条线、摄像头模块或机械细节。
- Keep decals visually subordinate to the vehicle. / 贴纸视觉上必须从属于车辆本身。
- Avoid symmetrical over-branding unless the part itself is paired. / 除非部件本身成对，否则避免对称式过度品牌化。
- Leave more empty space than expected. / 留出比预期更多的空白。
- Do not use stickers to cover or compete with factory badges. / 不要用贴纸遮挡或对抗原厂标识。
- Treat each sticker as a calibrated detail, not a decoration. / 将每张贴纸视为经过校准的细节，而不是装饰。
