# Panacea Tesla Model 3 Sticker Guide / Panacea Tesla Model 3 车贴指南

Target vehicle: Tesla Model 3, purchased in mainland China in September 2025. The supplied vehicle reference shows a deep metallic blue body color.

目标车型：Tesla Model 3，2025 年 9 月购于中国大陆。根据提供的车辆参考图，车身颜色为深蓝金属色。

This guide should become a complete design manual rather than a single rendering.

这份指南应发展为完整设计手册，而不是单张效果图。

## 01 Cover / 封面

Working title:

工作标题：

**Panacea Vehicle Identity Manual: Tesla Model 3 / Panacea 车辆识别手册：Tesla Model 3**

Tone: technical, editorial, restrained, product-manual-like.

气质：技术化、编辑化、克制，接近产品手册。

## 02 Philosophy / 理念

Core idea:

核心观点：

Vehicle identity should be discovered, not announced.

车辆识别应被发现，而不是被宣告。

The sticker system is a layer of architectural micro-branding. It should read like a product detail from a lab, archive, or design studio.

车贴系统是一层建筑化微型品牌识别。它应像来自实验室、档案馆或设计工作室的产品细节。

## 03 Vehicle Blueprint / 车辆蓝图

Required views:

所需视图：

- front / 正面
- rear / 后面
- driver side / 驾驶侧
- passenger side / 乘客侧
- top / 顶视图
- interior detail zones / 内饰细节区域

Blueprints should prioritize panel lines, lights, cameras, handles, ports, wheel geometry, and glass boundaries.

蓝图应优先标注钣金线、灯组、摄像头、门把手、接口、轮毂几何与玻璃边界。

Use `assets/vehicle/reference/tesla-model-3-2025-china-deep-blue-reference.png` as the current real-vehicle reference for color, trim contrast, glass darkness, and placement scale.

使用 `assets/vehicle/reference/tesla-model-3-2025-china-deep-blue-reference.png` 作为当前真实车辆参考，用于判断颜色、饰条对比、玻璃深度与贴附尺度。

## 08 Color Guide / 色彩指南

Map colors to vehicle surfaces:

将颜色映射到车辆表面：

| Vehicle Surface / 车辆表面 | Recommended Sticker Color / 推荐贴纸颜色 |
| --- | --- |
| Deep metallic blue paint / 深蓝金属车漆 | Matte silver first, matte white only for stronger contrast / 优先哑光银，需要更强对比时使用哑光白 |
| White paint / 白色车漆 | Gunmetal gray or matte silver / 枪灰或哑光银 |
| Black trim / 黑色饰条 | Matte white or matte silver / 哑光白或哑光银 |
| Glass / 玻璃 | Matte white / 哑光白 |
| Dark wheel / 深色轮毂 | Matte silver / 哑光银 |
| Interior dark plastic / 内饰深色塑料 | Matte silver or gunmetal gray / 哑光银或枪灰 |
