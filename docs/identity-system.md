# Panacea Identity System / Panacea 品牌识别系统

## Positioning / 品牌定位

`panacea` is not positioned as a normal consumer brand. It belongs closer to industrial design, product design, experimental lab work, objects, archive, research, editorial culture, and architecture.

`panacea` 不是一个普通消费品牌。它更接近工业设计、产品设计、实验室、物件、档案、研究、编辑文化与建筑。

The identity should feel precise, quiet, object-like, and durable. It should avoid campaign graphics, trend typography, lifestyle decoration, and loud branding.

这套识别系统应当精确、安静、物件化、耐久。它应避免活动海报式图形、潮流字体、生活方式装饰与高声量品牌露出。

## Logo System / Logo 系统

The current logo system uses an original geometric lettering language rather than a commercial typeface.

当前 Logo 系统使用原创几何字母语言，而不是商业字体。

Construction rules:

构成规则：

- Keep stroke weight unified across letters. / 所有字母保持统一线宽。
- Use 45-degree chamfers as the primary cut language. / 以 45 度切角作为主要切割语言。
- Preserve architectural, folded, modular letterforms. / 保留建筑化、折线化、模块化字形。
- Prefer open spaces and constructed counters. / 优先使用开放空间与构造式字腔。
- Avoid decorative curves. / 避免装饰性曲线。
- Treat the full `panacea` wordmark as one continuous symbol. / 将完整 `panacea` 字标视为一个连续符号。
- Do not over-emphasize the first letter. / 不要过度突出首字母。
- Preserve the current `a` icon as a core brand asset. / 保留当前 `a` 图标作为核心品牌资产。

The latest `panacea` wordmark has unified top chamfers across `p`, `a`, `n`, `a`, `c`, `e`, `a`, creating a continuous roofline. This is a system decision.

最新版 `panacea` 字标统一了 `p`、`a`、`n`、`a`、`c`、`e`、`a` 的顶部切角，形成连续屋檐线。这是系统级设计决策。

## Tone / 气质

Use these qualities as filters:

用以下关键词作为判断标准：

- minimal / 极简
- architectural / 建筑化
- industrial / 工业感
- geometric / 几何化
- editorial / 编辑感
- museum-like / 博物馆感
- gallery-like / 画廊感
- timeless / 永恒
- quiet luxury / 安静的高级感
- discoverable / 可被发现

Reject these qualities:

拒绝以下方向：

- racing graphics / 赛车拉花
- decoration-first layout / 装饰优先的版式
- chrome effects / 镀铬效果
- reflective novelty / 反光噱头
- red accent dependency / 依赖红色点缀
- large surface coverage / 大面积覆盖
- unrelated typography / 无关字体
- mascot or lifestyle illustration / 吉祥物或生活方式插画

## Application Principle / 应用原则

Every application must strengthen the whole system. Vehicle stickers, packaging, website UI, product tags, light boxes, and software icons should feel like different outputs of the same design logic.

每一个应用都必须强化整个系统。车贴、包装、网站 UI、产品标签、灯箱与软件图标，都应像同一套设计逻辑的不同输出。

The brand should be recognized through proportion, restraint, placement, material, and construction before scale or repetition.

品牌识别应首先通过比例、克制、位置、材料与构成被感知，而不是依赖尺寸或重复。
